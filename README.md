# GSM Firmware CDN Assets

This repository hosts static assets for [gsmfirmware.net](https://www.gsmfirmware.net/) served via CDN at `cdn.gsmfirmware.net`.

## Contents
- **Favicons** - Site icons for browsers and mobile devices
- **OG Images** - Open Graph images for social sharing
- **Logo** - Site logo in various formats
- **Manifest.json** - PWA manifest file

## Usage
Assets are served via CDN with the base URL:
```
https://cdn.gsmfirmware.net/[file-path]
```
